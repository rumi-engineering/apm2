# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/Anveio/apm2/security/advisories/new).

- We aim to acknowledge reports within 48 hours
- We coordinate disclosure timeline with reporters
- Public disclosure after patch is available

## Security Response Times

| Severity | Response |
|----------|----------|
| Critical | 24 hours |
| High | 7 days |
| Medium | 30 days |
